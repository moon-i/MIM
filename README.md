# I'm J (MIM) 
MBTI J인 계획 러버들을 위한 계획 도우미 Application
<br>
<br>

## Download
심사중..
<br>
<br>

## Features
<li> 오늘의 계획 실행하기 - 계획을 하나하나 지우며 달성하세요 </li>
<li> 계획 작성하기 - 달력을 이용하여 원하는 날짜에 계획을 추가하세요. </li>
<li> 태그 생성하기 - 계획을을 그룹지을 수 있도록 태그를 생성하세요. </li>
<li> 통계 확인하기 - 계획 달성을 통계로 확인하여 동기부여를 받아보세요! </li>
